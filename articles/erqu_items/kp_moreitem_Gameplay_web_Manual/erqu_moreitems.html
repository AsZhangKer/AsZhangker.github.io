合成路线指南

合成路线总览：

第一阶段：基础强化

  • 目标：制作强化合金锭 → 强化下界合金块
  • 材料：下界合金锭（高炉烧炼获得）
  • 用途：所有合成的基础材料

第二阶段：星核材料

  • 目标：制作下界之星核心
  • 材料：8下界之星 + 强化下界合金块
  • 用途：星核力量的基础

第三阶段：末地材料

  • 目标：末地精髓 → 末地合金锭 → 末地合金块
  • 材料：龙首、鞘翅、下界之星核心
  • 用途：末地力量的装备材料

第四阶段：虚空材料

  • 目标：虚空精髓 → 虚空材料
  • 材料：末地水晶、末地精髓、末地合金块
  • 用途：终极装备制作材料
